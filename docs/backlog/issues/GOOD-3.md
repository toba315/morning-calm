## Description

ペルソナ名称：ノイズに疲れつつも、ついスマホを見てしまう「受動的マインド層」

1. 基本プロフィール
25-29歳。都市部で働く会社員。単身者。
毎朝決まった時間の電車で通勤する。情報収集に対して特別な熱意（スキルアップや資格勉強など）があるわけではなく、日々の生活や仕事を平穏にこなすことを重視する。

スマホの利用実態：
電車内では特に目的もなくスマホを触る。XやInstagram、一般的なニュースアプリを惰性でスクロールしている。
しかし、流れてくる事件のニュースや、タイムラインの攻撃的な言葉、画面を遮る動画広告に対して、心の中で「またこれか」「見なければよかった」と小さなストレスを溜めている。

2. アプリの利用コンテキスト（いつ、どのように使うか）
瞬間：朝の通勤電車、吊り革に掴まりながらスマホを片手で持っている時間。
動機（インサイト）：「SNSのギスギスした空気は見たくない。かといって、何も見ないで満員電車に乗っているのは退屈。短時間でサクッと読めて、絶対に嫌な気持ちにならない、ちょうどいい暇つぶしが欲しい」
行動パターン：アプリを開き、厳選された5本のポジティブニュースを数分でサクッと読み終える。「今日も良いニュースがあったな」という軽い安心感を得て、アプリを閉じる。

3. アプリへの期待値と受容性
金銭感覚：
アプリの機能に対してお金を払う意思は極めて低い。月額制や買い切りを求められた時点で利用をやめる。

広告への許容度：
5本のニュースを無料で読むためであれば、広告の存在自体は許容する。ただし、スクロールを邪魔するポップアップ広告や、不快な映像が流れる動画広告は強く拒絶する。
画面下部に静かに佇むような、デザインを損なわない「環境型広告」であれば、ノイズとして認識せず受け入れる。
