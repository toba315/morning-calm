## Description

Morning Calm（モーニング・カーム）
精神的ノイズをゼロにする、世界で一番優しい「朝のメンタル・チューニング」習慣化アプリ。

1. ターゲット
朝を前向きな気持ちでスタートさせたいが、SNSや刺激の強いニュースを見てダラダラ過ごしてしまうビジネスパーソン。
感受性が高く、朝一番に「優しい世界（ポジティブなファクト）」に触れることで、心の平穏を保ちたい層。

2. コア・ベネフィット（提供する価値）
過度な依存や「無限スクロール」を誘発する設計を徹底排除。朝の5分〜10分でサクッと完了し、すっきりと1日を始められる「非依存型」の良質な習慣を提供。
疲れている時でも、アプリを開いて「前向きな事実」に数分触れるだけで、認知がポジティブに切り替わり、「今日も大丈夫」という活力を自然に得られる。
広告の煽りや他人の批判コメント、ネガティブなタイムラインから完全に隔離された、朝一番の「聖域（サードプレイス）」としての安心感。

3. 独自性（独自の手段・根拠）
一般的な「タスクをチェックする」習慣化アプリとは異なり、「厳選されたプラスの事象（誕生、発見、貢献などのファクト）」を数本読むこと自体を朝の行動トリガー（習慣）とする設計。
ユーザーをアプリに引き止めないため、1日の配信数や利用時間をあえて制限。「読み終えたら、アプリを閉じて現実の世界へ行く」ことを促す、自立支援型のUI/UX。
広告、コメント欄、Bad評価、他ユーザーとの比較機能を徹底排除。ノイズにならない広告（後述）を除き、画面全体を「心を整える瞑想空間」のようにデザインする。

4. ビジネスモデルの方向性
ハイブリッド・フリーミアム
無料ユーザー：1日10分までの利用、または基本ルーティン（1日5本のポジティブ・インプット）の体験。
有料ユーザー：月額少額（コーヒー1杯分程度）。習慣化を支えるログ機能（カレンダー記録、過去に元気をもらったニュースのストック機能など）の解放。
