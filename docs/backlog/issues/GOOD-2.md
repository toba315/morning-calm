## Description

*ポジティブニュースの定義
本プロジェクトにおけるポジティブとは、単なる「楽しいニュース」のみではない。
「未来への可能性、あるいは生命・社会の前進を感じさせ、読者に前向きな行動のきっかけ（ヒント）を与える事実」を定義の核とする。

**採用するポジティブの3分類
ポジティブニュースを選別・配信する際、以下のいずれかに該当することを条件とする。

 ''A. 創造と誕生（希望の創出）'' 
無から有が生じ、純粋に生命力や新しい価値を感じさせる事象。（例：動物の赤ちゃん誕生、新種の発見、地域活性化に繋がる施設の新規開業）

 ''B. 進歩と克服（知恵による前進）'' 
知恵、努力、技術によって困難を乗り越え、未来を切り拓いた事象。（例：難病の治療法確立、技術革新の成功、災害からの復興完了、自己記録の更新）

 ''C. 貢献と調和（善意の連鎖）'' 
善意や勇気によって、他者や環境が救われ、あるいは調和が生まれた事象。（例：迷子の救助、国境を越えた支援、歴史的な和解、地域社会での善行）

**ポジティブ判定の必須条件
本アプリの独自価値（USP）を担保するため、以下の2点を満たすもののみを掲載対象とする。

 ''成長・ヒントの含有'' 
読後に「自分もこうありたい」「明日からこれを意識しよう」と思わせる、具体的な行動原理や思考のヒントが含まれていること。

 ''事実（Fact）ベース'' 
主観的な感想や出所不明の美談を排除し、公的に確認された客観的事実であること。

**厳格な排除基準（ネガティブ・フィルター）
たとえ結果が「平和」や「解決」であっても、以下の要素が含まれる場合は非ポジティブとみなし、一律で排除する。

 ''負の清算'' 
犯人逮捕、判決、鎮圧、事故の収束など。事象の起点が「悪意」や「過失」にあるものは、解決のニュースであっても朝の平穏を乱すため採用しない。

 ''対立と批判'' 
論破、解任、他者の否定、または相対的な優位性によって得られた成果は排除する。

 ''過度な恐怖の想起'' 
凄惨な事件や事故を強く連想させる背景が主役となっているものは採用しない。ただし、焦点が「未来の希望」に完全にある復興ニュースなどはこの限りではない。
