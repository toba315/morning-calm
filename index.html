    ニュース収集

    公式RSSまたは公式新着ページから、日付・タイトル・URL・発表元だけを取得します。 本文の丸ごと保存や要約ではなく、確認できるファクトから短いオリジナル記事を生成します。

    実行: npm run news:fetch

    取得対象

    • 水族館・動物園・博物館などの公式お知らせ
    • 研究機関・自治体・政府機関の一次発表
    • 本文流用なしで扱える、明るい事実ベースの話題

    著作権対策

    1. 記事本文は保存しない
    2. RSS descriptionは本文生成に使わない
    3. タイトル・日付・URL・発表元からファクトを整理
    4. アプリ文面は独自テンプレートで新規作成
    5. 詳細は公式発表へリンクする

    生成ファイル

    scripts/fetch-news.mjs が news-data.js を生成し、 画面は window.MORNING_CALM_NEWS を読み込みます。
